5. Pattern Saving, Loading, and CHAIN

This page explains how to manage patterns during normal music use.

Save Your Current Pattern

  1. Open System Settings (encoder long press).
  2. Select Save Pattern.
  3. The current state (steps and parameters) is stored.

Load A Pattern

  1. Open System Settings.
  2. Select Load Pattern.
  3. Rotate encoder to choose a pattern.
  4. Press encoder short or medium to load it.

Create A New Pattern

  1. Open System Settings.
  2. Select New Pattern.
  3. A new pattern name is created automatically.

Delete A Pattern

  1. Open System Settings.
  2. Select Delete Pattern.
  3. Rotate to choose pattern.
  4. Press encoder medium to delete.

Use CHAIN In Practice

CHAIN is ideal for creating longer arrangements from multiple pattern slots.

  1. Enter edit mode and open Edit Track popup.
  2. Select CHAIN entry.
  3. Set ON/OFF to ON.
  4. Set Lx to how many slots you want in sequence.
  5. Set Px to choose which slot you are editing now.
  6. Edit each slot and save if needed.

Simple Arrangement Example

  • P1: basic groove
  • P2: add extra snare fills
  • P3: lower PROBABILITY on hats for variation
  • Set CHAIN ON and L3 to cycle P1 -> P2 -> P3

5. Scherm: System Settings

Hier beheer je patroonacties, sample set, display-opties en herstart/exit.

Schermweergave

+--------------------------+
| SYSTEM SETTINGS          |
| SSID: Groovebox          |
| IP: 192.168.4.1          |
| MAC: aa:bb:cc:dd:ee:ff   |
|>Add Pattern<             |
| Delete Pattern           |
| Card Storage             |
| Load Sample Set          |
| Erase WiFi credentials   |
| Start WiFiManager        |
| Set Theme (...)          |
| Rotate Display (...)     |
| Encoder Order (...)      |
| Restart Groovebox        |
| Exit                     |
+--------------------------+

Bediening

  • Open/sluit: encoder lang vanaf Groovebox.
  • Encoder draaien: regel selecteren.
  • Encoder kort/medium: geselecteerde actie uitvoeren/openen.
  • Knop2 kort: één niveau terug of Settings sluiten.

Wat belangrijk is voor muziek maken

  • Add Pattern: extra patroonslot voor variaties binnen je set.
  • Delete Pattern: verwijdert een patroon dat je niet meer gebruikt.
  • Card Storage: patroonset laden/opslaan/hernoemen/kopiëren.
  • Load Sample Set: wisselt klankset; bestaande grooves krijgen direct ander karakter.
  • Restart Groovebox: snelle herstart bij live issues.

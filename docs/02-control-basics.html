2. Scherm: Track Edit (stappen en attributen)

In dit scherm wijzig je per track en per step hoe het patroon klinkt.

Schermweergave (voorbeeld in edit mode)

+--------------------------+
| GROOVEBOX                |
| BPM 120 SW 08 PLAY p01   |
|>KICK   x---[x]---x---x---|
| SNARE  ----x-------x---  |
| CH     x-x-x-x-x-x-x-x-  |
| OH     -------x--------  |
| TONE   ----------------  |
| VEL 120                  |
| TR1 S:05                 |
+--------------------------+

Bediening in Track Edit

  • Encoder draaien: wijzigt waarde op actieve pagina.
  • Encoder kort: togglet/activeert functie op actieve pagina.
  • Encoder medium: opent Edit Track popup.
  • Encoder lang: vorige parameterpagina (cyclisch).
  • Knop2 kort: verlaat edit mode.

Track/Step-attributen en hoorbaar effect

TRIG (step aan/uit)

  • Wat stel je in: of de step triggert op die positie.
  • Hoorbaar: ritmepatroon zelf; meer actieve steps = drukker, minder = opener.

VEL (velocity per step)

  • Wat stel je in: aanslagsterkte van de step.
  • Hoorbaar: accent en dynamiek; hoge VEL komt naar voren, lage VEL zakt naar achteren.

PITCH (pitch lock per step)

  • Wat stel je in: toonhoogte-offset per step.
  • Hoorbaar: variatie in toon; subtiel voor beweging, extreem voor effecten/fills.

DECAY (decay lock per step)

  • Wat stel je in: lengte/uitklank van de hit per step.
  • Hoorbaar: korte decay = strakker/percussiever; lange decay = voller/meer staart.

PROB (probability per step)

  • Wat stel je in: kans dat de step echt speelt (0-100%).
  • Hoorbaar: variatie over loops; lagere kans maakt patronen minder statisch.

MUTE (trackbreed)

  • Wat stel je in: track uit/aan zonder steps te verwijderen.
  • Hoorbaar: directe arrangementwissel, handig voor breaks en opbouw.

CHAIN (patternvolgorde)

  • Wat stel je in: chain aan/uit en lengte van patroonreeks.
  • Hoorbaar: langere muzikale structuur in plaats van één korte loop.

2. Control Basics

This page explains the two physical controls you use all the time: the encoder and Knop2.

Encoder

  • Rotate left/right: move selection or change a value.
  • Short press: open edit, toggle a step, or confirm selection.
  • Medium press: open popups.
  • Long press: open/close System Settings.

Knop2

  • Short press: Play/Stop in normal mode, or exit edit mode.
  • Medium press: open Tempo Edit popup (BPM selected).
  • Long press: open Tempo Edit popup (SW selected).

Normal Mode vs Edit Mode

  • Normal mode: you mainly select tracks and start/stop playback.
  • Edit mode: you change steps and parameters for the selected track.
  • You enter edit mode with encoder short press.
  • You leave edit mode with Knop2 short press.

Tip For Beginners

If something feels confusing, press Knop2 short once to exit edit mode and return to a simpler interaction state.

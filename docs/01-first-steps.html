1. Scherm: Groovebox (hoofdscherm)

Dit is je speelscherm: tracks kiezen, play/stop, en live overzicht van je patroon.

Schermweergave

+--------------------------+
| GROOVEBOX                |
| BPM 120 SW 08 PLAY p01   |
|>KICK   x---x---x---x--- <|
| SNARE  ----x-------x---  |
| CH     x-x-x-x-x-x-x-x-  |
| OH     -------x--------  |
| TONE   ----------------  |
| METAL  ----------------  |
| STEP 05 CUR 05 p01->p02  |
+--------------------------+

Wat je hier bedient

  • Encoder draaien: selecteert track (KICK/SNARE/CH/OH/TONE/METAL).
  • Encoder kort: ga naar Track Edit op de gekozen track.
  • Encoder medium: open Tempo Edit popup.
  • Encoder lang: open/sluit System Settings.
  • Knop2 kort: PLAY/STOP.
  • Knop2 medium: open Tempo Edit met BPM geselecteerd.
  • Knop2 lang: open Tempo Edit met SW geselecteerd.

Wat je als luisteraar merkt

  • BPM omhoog: meer energie en snelheid.
  • SW omhoog: meer shuffle/swing, minder strak recht.
  • Trackselectie: bepaalt welke klank je straks bewerkt.
  • PLAY/STOP: directe start/stop van de groove.

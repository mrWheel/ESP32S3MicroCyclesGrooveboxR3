1. First Steps: Make Your First Beat

This page is for first-time users. Follow these steps in order and you will have a running rhythm in a few minutes.

Before You Start

  • Turn the Groovebox on.
  • Wait until the main [Groovebox] screen appears.
  • Make sure your speaker or headphones are connected.

Character Preview: What You Should See First

        +--------------------------+
        | GROOVEBOX                |
        | BPM 120  SW 08  STOP     |
        |>KICK   x---x---x---x--- <|
        | SNARE  ----x-------x---  |
        |  ...                     |
        | STEP 01 CUR 01 P1 V0     |
        +--------------------------+

Make a Basic Beat

  1. Press Knop2 short once to start playback.
  2. Rotate encoder to select KICK.
  3. Press encoder short to enter edit mode.
  4. On TRIG page, rotate to move the step cursor.
  5. Press encoder short to turn steps on or off.
  6. Add hits on steps 1, 5, 9, and 13 for a classic kick rhythm.
  7. Press Knop2 short to exit KICK edit mode.
  8. Rotate encoder to select SNARE.
  9. Press encoder short to enter edit mode again.
  10. On TRIG page, add hits on steps 5 and 13.
  11. Press Knop2 short to exit edit mode.

What The Buttons Mean (Quick View)

  • Encoder rotate: select track (normal) or edit values (edit mode).
  • Encoder short: enter edit mode, toggle steps, or confirm small actions.
  • Encoder medium: open popup (Tempo Edit or Edit Track popup).
  • Encoder long: open or close System Settings.
  • Knop2 short: Play/Stop, or exit edit mode.
  • Knop2 medium: open Tempo Edit popup with BPM selected.
  • Knop2 long: open Tempo Edit popup with SW selected.

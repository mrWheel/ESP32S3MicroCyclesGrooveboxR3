Introduction

Welcome to the ESP32 MicroCycles Groovebox.

This instrument is designed around a simple idea:

  • Load a Pattern Group
  • Create patterns
  • Chain patterns into songs
  • Save your work
  • Perform live

This manual focuses on making music, not programming.

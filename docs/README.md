## Groovebox Summary

- 6 tracks: Kick, Snare, CH, OH, Tone, Metal
- 16-step sequencer
- Encoder + encoder button + KEY0 workflow
- ST7789 320x240 screen
- I2S audio output to DAC/amplifier

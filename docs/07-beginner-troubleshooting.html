7. Beginner Troubleshooting

These checks are written for normal users while operating the Groovebox.

I Hear Nothing

  1. Check if transport says STOP. Press Knop2 short to start PLAY.
  2. Check MUTE page for muted tracks.
  3. Check TRIG page and confirm there are active steps.
  4. Try another track (for example KICK or SNARE).

Pattern Sounds Too Repetitive

  • Lower PROBABILITY on a few steps (for example from 100% to 70%).
  • Vary VELOCITY values between steps.
  • Add slight SW amount for groove feel.

Pattern Feels Too Busy

  • Mute one track and listen again.
  • Reduce number of TRIG steps.
  • Lower VELOCITY on secondary hits.

CHAIN Does Not Change Pattern

  • Open CHAIN and verify ON/OFF is set to ON.
  • Set Lx to at least 2.
  • Use Px to select each slot and ensure slots actually differ.

I Am Lost In Menus

  • Press Knop2 short to back out of edit contexts.
  • Close settings with encoder long press.
  • Return to [Groovebox] main screen and start from TRIG page.

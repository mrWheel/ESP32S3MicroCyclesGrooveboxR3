7. Scherm: Bevestigings- en statusschermen

Deze schermen geven zekerheid bij kritieke acties zoals wissen, WiFi-acties en opslaan.

Schermweergave: Yes/No bevestiging

+--------------------------+
| ERASE WIFI CREDENTIALS   |
|       Are you sure?      |
|          >No<            |
|           Yes            |
+--------------------------+

Schermweergave: WiFi/status scherm

+--------------------------+
| WIFI PORTAL STARTED      |
| Connect to Groovebox     |
| Enter credentials        |
| Waiting...               |
+--------------------------+

Schermweergave: korte statuspopup

+--------------------------+
| Saved group A            |
|                          |
| (verdwijnt automatisch)  |
+--------------------------+

Bediening

  • Encoder draaien: wisselt keuze (No/Yes) in bevestigingen.
  • Encoder kort/medium: bevestigt geselecteerde keuze.
  • Knop2 kort: annuleert/bevestigingsscherm sluiten waar mogelijk.

Praktische eindgebruikersregels

  • Gebruik No als je twijfelt; je kunt altijd opnieuw kiezen.
  • Verwijderen is bedoeld voor opruimen; check patternnaam eerst goed.
  • Bij "Waiting..." laat je het scherm staan tot de actie klaar is.

Snelle hulp bij "ik hoor niets"

  1. Ga naar Groovebox en druk Knop2 kort (PLAY).
  2. Controleer in Track Edit of TRIG-steps actief zijn.
  3. Controleer MUTE (track mag niet op mute staan).
  4. Zet PROB op 100% tijdens testen.
  5. Zet VEL hoger op de actieve steps.

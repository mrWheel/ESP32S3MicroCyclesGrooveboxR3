Chapter 11

Chain

Chain creates song structure.

Example:


P01 -> P02
P02 -> P03
P03 -> P04
P04 -> P01

Result:


P01
P02
P03
P04
repeat

Another example:


P01 -> P02
P02 -> P03
P03 -> P02

Creates an endless groove.

4. Scherm: Edit Track popup

Deze popup geeft snelle toegang tot alle bewerkpagina's van de geselecteerde track.

Schermweergave (selectie)

+------------------------+
| EDIT TRACK             |
| >VELOCITY< 120         |
|  PITCH      +00        |
|  DECAY      100%       |
|  PROBABILITY 100%      |
|  MUTE       OFF        |
|  CHAIN      ON L3 p02  |
+------------------------+

Schermweergave (waarde-edit met CHAIN focus)

+------------------------+
| EDIT TRACK             |
| CHAIN >ON< L3 p02      |
| CHAIN ON >L3< p02      |
| CHAIN ON L3 >p02<      |
| kort = volgende focus  |
| draai = waarde         |
+------------------------+

Bediening

  • Openen: encoder medium terwijl je in Track Edit bent.
  • Encoder draaien: regel kiezen, of waarde wijzigen in value-edit.
  • Encoder kort: value-edit aan/uit; bij CHAIN wisselt focus ON/OFF -> L -> P.
  • Encoder medium: terug naar selectie (value-edit uit).
  • Encoder lang: popup sluiten.
  • Knop2 kort/medium/lang: popup sluiten.

Attribuutfocus en luistereffect

  • VELOCITY: groove-accenten en dynamiek.
  • PITCH: toonvariatie per hit (fills, beweging, spanning).
  • DECAY: kort/lang karakter per hit (strak vs. ruim).
  • PROBABILITY: gecontroleerde variatie per loop.
  • MUTE: live arrangement-schakelaar.
  • CHAIN: volgorde van patronen voor langere composities.

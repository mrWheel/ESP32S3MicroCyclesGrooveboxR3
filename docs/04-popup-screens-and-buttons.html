4. Popup Screens and Their Buttons

This page explains every popup you can open from the Groovebox screen.

Tempo Edit Popup

Open with encoder medium (normal mode), Knop2 medium, or Knop2 long.

  • Rotate encoder: change selected value.
  • Short press encoder: switch selection between BPM and SW.
  • Medium press encoder: close popup.
  • Knop2 short/medium/long: close popup.

Character Preview: Tempo Edit Popup

        +------------------------+
        | TEMPO EDIT             |
        | BPM >120<              |
        | SW   08                |
        | rotate = change.       |
        | short  = switch BPM/SW |
        +------------------------+

Edit Track Popup

Open with encoder medium while already in edit mode.

Entries: VELOCITY, PITCH, DECAY, PROBABILITY, MUTE, CHAIN.

Character Preview: Edit Track Popup (Selection)

        +------------------------+
        | EDIT TRACK             |
        | >VELOCITY< 127         |
        |  PITCH      +00        |
        |  DECAY      100%       |
        |  PROBABILITY 100%      |
        |  MUTE       OFF        |
        |  CHAIN      ON L3 P2   |
        +------------------------+

Selection State

  • Rotate encoder: move popup selection.
  • Short press encoder: enter value-edit for selected entry.
  • Medium press encoder: leave value-edit and remain in popup.
  • Long press encoder: close popup.

Value-Edit State

  • VELOCITY/PITCH/DECAY/PROBABILITY: rotate to edit value.
  • MUTE: rotate toggles ON/OFF.
  • CHAIN: short press cycles focus ON/OFF -> Lx -> Px, rotate edits focused field.

Character Preview: CHAIN Focus In Value-Edit

        +----------------------+
        | F1: CHAIN >ON< L3 P2 |
        | F2: CHAIN ON >L3< P2 |
        | F3: CHAIN ON L3 >P2< |
        |                      |
        | short  = next focus  |
        | rotate = edit field  |
        +----------------------+

CHAIN Fields In Popup

  • ON/OFF: enables or disables chain playback.
  • Lx: chain length (how many pattern slots are in the chain).
  • Px: active chain slot (which slot you are editing now).

System Settings Screen

Open or close with encoder long press.

  • Rotate encoder: move selection in the list.
  • Short or medium press encoder: open or execute selected item.
  • Knop2 short: back/close current settings layer.

Character Preview: System Settings

        +-------------------------+
        | SYSTEM SETTINGS         |
        | SSID: MyWifi            |
        | IP: 192.168.1.44        |
        |>Load Pattern<           |
        | Save Pattern            |
        | New Pattern             |
        | Delete Pattern          |
        | Exit                    |
        +-------------------------+

Pattern List Popup (Load/Delete)

  • Rotate encoder: choose a pattern name.
  • Load mode: short or medium press loads selected pattern.
  • Delete mode: medium press deletes selected pattern.
  • Knop2 short: back to System Settings.

Confirmation Popups (Yes/No)

  • Rotate encoder: switch between No and Yes.
  • Short or medium press encoder: confirm selected choice.
  • Knop2 short: cancel and go back.

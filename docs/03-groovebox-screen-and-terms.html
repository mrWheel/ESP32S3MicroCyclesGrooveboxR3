3. Groovebox Screen and Terms

The [Groovebox] screen is your main working screen. Most music-making happens here.

What You See On The Groovebox Screen

  • Header: BPM, SW, and transport state (PLAY or STOP).
  • Track rows: KICK, SNARE, CH, OH, TONE, METAL.
  • Parameter/info row: context text in edit mode.
  • Footer: current playing step, cursor step, active pattern slot, and voice activity.

Buttons On This Screen (Normal Mode)

  • Encoder rotate: select another track.
  • Encoder short: enter edit mode for the selected track.
  • Encoder medium: open Tempo Edit popup.
  • Encoder long: open/close System Settings.
  • Knop2 short: Play/Stop.
  • Knop2 medium: open Tempo Edit popup with BPM selected.
  • Knop2 long: open Tempo Edit popup with SW selected.

Buttons On This Screen (Edit Mode)

  • Encoder rotate: action depends on the selected page (TRIG, VEL, PITCH, DECAY, PROB, MUTE, CHAIN).
  • Encoder short: toggle or confirm action for current page.
  • Encoder medium: open Edit Track popup.
  • Encoder long: move to previous parameter page.
  • Knop2 short: leave edit mode.

Main Edit Pages (What They Mean)

TRIG

  • What it is: step on/off pattern for the selected track.
  • What it does: decides where the sound is played.
  • How to use: rotate to move cursor, short press to toggle step.

VEL (VELOCITY)

  • What it is: step volume strength (how hard a step hits).
  • What it does: higher VELOCITY sounds stronger; lower VELOCITY sounds softer.
  • How to use: in VEL page, rotate to change VELOCITY for the cursor step.

PITCH

  • What it is: per-step pitch lock value.
  • What it does: stores tonal offset for that step.
  • How to use: rotate to change PITCH value, short press to toggle lock on/off.

DECAY

  • What it is: per-step decay lock amount.
  • What it does: DECAY lock changes the strength of that step.
  • How to use: rotate to change DECAY percentage, short press to toggle lock on/off.

PROB (PROBABILITY)

  • What it is: chance (0% to 100%) that a step will trigger.
  • What it does: creates variation and less repetitive loops.
  • How to use: rotate to set PROBABILITY for the cursor step.

MUTE

  • What it is: track mute state.
  • What it does: muted track stays in pattern but makes no sound.
  • How to use: rotate to select track, short press to toggle MUTE.

CHAIN

  • What it is: pattern chaining controls.
  • What it does: plays multiple pattern slots in sequence.
  • How to use: rotate to change chain length, short press to toggle CHAIN on/off.

Header Terms Explained

  • BPM: Beats Per Minute. Higher BPM means faster tempo.
  • SW: Swing amount. It shifts timing feel between straight and shuffled groove.

Practical Beginner Example

  1. Start with BPM 120 and SW 0 for a straight rhythm.
  2. Add TRIG steps first.
  3. Then use VELOCITY and PROBABILITY for groove.
  4. Use MUTE for live arrangement changes.
  5. Use CHAIN to play multiple pattern slots automatically.

6. How to Use Parameters Musically

You do not need music theory to get good results. Use the ideas below as a practical starting point.

BPM

  • Lower BPM: slower groove, more space.
  • Higher BPM: more energy and urgency.
  • Start around 110 to 125 if you are unsure.

SW (Swing)

  • SW adds shuffle feel by changing timing balance between steps.
  • SW 0 is straight and tight.
  • Try SW 6 to 14 for a human groove.

VELOCITY

  • Use different VELOCITY values to avoid robotic patterns.
  • Example: hats at 120, 90, 110, 80 for movement.

PITCH

  • Use small pitch offsets on selected steps for variation.
  • Try subtle values first (for example -2 to +2).

DECAY

  • Shorter decay-like values can make hits feel tighter.
  • Higher values can make accents feel fuller.

PROBABILITY

  • 100% means the step always plays.
  • 50% means it plays about half the time.
  • Great for hi-hats, fills, and evolving loops.

MUTE

  • Use MUTE to bring tracks in and out live.
  • Mute is fast and safe for arrangement changes during playback.

CHAIN

  • CHAIN links pattern slots into a longer sequence.
  • Use CHAIN for verse/variation style transitions without stopping.

Beginner Workflow

  1. Create the TRIG pattern first.
  2. Set BPM and SW for overall feel.
  3. Add VELOCITY variation.
  4. Add PROBABILITY to one or two tracks.
  5. Use MUTE live, then create CHAIN for longer playback.

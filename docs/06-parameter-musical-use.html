6. Scherm: Pattern- en Sample-lijsten

Dit zijn selectieschermen voor laden/verwijderen van patronen en het kiezen van sample sets.

Schermweergave: Load/Delete Pattern

+--------------------------+
| LOAD PATTERN             |
|>Local: A01               |
| Local: A02 -> A03        |
| Card:  B01               |
| Card:  B02 -> B03        |
| ...                      |
+--------------------------+

Schermweergave: Card Storage

+--------------------------+
| CARD STORAGE             |
|>Load Pattern             |
| Save Pattern             |
| Rename Pattern           |
| Copy Pattern             |
| Exit                     |
+--------------------------+

Schermweergave: Load Sample Set

+--------------------------+
| LOAD SAMPLE SET          |
|>RCK                      |
| HSE                      |
| TEC                      |
| ...                      |
+--------------------------+

Bediening

  • Encoder draaien: door de lijst bladeren.
  • Encoder kort/medium: geselecteerd item uitvoeren (laden/verwijderen/openen).
  • Knop2 kort: terug naar vorig scherm.

Hoorbaar effect in de praktijk

  • Ander pattern laden: direct andere ritmische inhoud en arrangement.
  • Pattern met chain-target: automatische overgang naar volgend pattern.
  • Andere sample set: zelfde sequence, compleet andere klankkleur.

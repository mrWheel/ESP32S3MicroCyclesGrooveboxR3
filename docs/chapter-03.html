Understanding Patterns

A Pattern is one musical section.

Examples:


P01 Intro
P02 Groove
P03 Breakdown
P04 Drop

Think of each pattern as a building block.

A complete song may contain:


P01 Intro
P02 Groove
P03 Groove Variation
P04 Breakdown
P05 Build
P06 Drop

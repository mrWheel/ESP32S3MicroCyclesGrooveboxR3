HOUSEJAM Groovebox Pattern Group

Copy this folder to the SD card as:

/patterns/HOUSEJAM/

Chain:
p01 -> p02 -> p03 -> p04 -> p05 -> p03

p06 is intentionally outside the chain and can be used as the drop/final-stop pattern.

This set uses many per-step variables:
trigger, velocity, probability, lockPitch, lockDecay, microTiming, flam, accent, ratchet, reverse, sampleStart, sampleEnd, pan, chokeGroup.
